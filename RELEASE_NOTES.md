## English

This is the first clean public release of `HaoXiang Document Assistant`, built with an Electron + React frontend and a FastAPI document-processing backend.

Downloads:

- `official-document-ai-assistant-offline-1.4.7-x64.exe`: offline Windows installer with local Ollama only.
- `official-document-ai-assistant-offline-1.4.7-x64.msi`: offline Windows MSI installer.
- `official-document-ai-assistant-online-1.4.7-x64.exe`: online-mode Windows installer.
- `official-document-ai-assistant-online-1.4.7-x64.msi`: online-mode Windows MSI installer.
- `official-document-ai-assistant-offline-1.4.7-x64.zip`: offline portable application archive.
- `official-document-ai-assistant-online-1.4.7-x64.zip`: online portable application archive.
- `SHA256SUMS.txt`: SHA-256 checksums for all release files.

This release does not bundle third-party font binaries. Obtain and install required fonts from properly licensed sources.
